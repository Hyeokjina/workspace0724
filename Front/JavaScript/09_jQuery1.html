jQuery

jQuery란

기존의 자바스크림트 언어만으로 구현하기에는 복잡했던 부분을 간소화하기위해 만들어진
javascript기반의 라이므러리
즉, javascript로 유용한 함수나 내용들을 다른 누군가가 이미 정의해 놓은것
복잡한 JS구문보다 짧고 쉽게 개발이 가능
==> 장점
1) DOM요소와 관련된 스크립트를 보다 쉽게 구현할 수 있음
2) AJAX(비동기통신)이벤트 처리등을 폭넓게 지원함
3) jQuery와 관련된 확장 플러그인, 오픈API등을 지원(치트,슬라이드,캘린더, 드래그앤드랍, ...)


jQuery라이브러리 연결

jQuery를 포함한 모든 외부자료는 2가지 방법으로 연결할 수 있음.

1.jQuery라이브러리 파일을 직접 다운로드 받은 후 파일 경로를 통해서 html연결.
2.CDN(content Dlivery network)방식
파일을 다운로드 받아서 저장할 필요 없이 라이브러리를 제공하는 사이트의 url을 제시하여
브라우저에서 html을 로딩 할 해당 서버에서 다운로드해 사용핼 수 있도록 하는 방식

jQuery구문 작성하기

jquery구문으로 DOM요소를 다루는 구문을 기본적으로 작성할 것이기 때문에
문서상에 요소들이 다만들어지고나서 실행되어야 한다.
